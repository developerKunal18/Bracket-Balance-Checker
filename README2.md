# Bracket Balance Checker

A Python script that checks whether brackets in an expression are balanced.

## Description
This project verifies whether all opening brackets have matching closing brackets in the correct order.

Supported brackets:
- ()
- []
- {}

## Features
- Stack-based logic
- Supports multiple bracket types
- Beginner-friendly implementation

## Technologies Used
- Python

## How to Run
1. Clone the repository
2. Navigate to the project folder
3. Run:

   python bracket_checker.py

4. Enter an expression when prompted

## Example

Input:
{[()]}

Output:
Balanced
